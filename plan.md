# 다해조 심부름센터 임무관리 앱

## 앱 목적
- **얼굴 인식 기반 직원 인증** → 임무 관리 시스템 접근
- web/play 프로젝트(다해조 채용 웹사이트)와 디자인 통일

## 주요 기능
1. **직원 인증**: 얼굴 인식으로 본인 확인
2. **현재 임무 확인**: 진행중인 임무 상세 정보 표시 (목록 아님, 단일 임무)
3. **임무 기록**: 사진/영상 촬영 및 저장
4. **미디어 확인**: 저장된 사진/영상 팝업으로 확인

## 다음 작업

### 4단계: MainScreen 개편

#### 목표
- 임무 **목록** 제거 → **현재 진행 임무 1개**만 표시
- 임무 내용은 서버 연동 전까지 더미 데이터 사용

#### UI 구성
```
┌─────────────────────────────────┐
│  다해조 심부름센터      [로그아웃] │
├─────────────────────────────────┤
│  ┌─────────────────────────┐    │
│  │ 인증된 직원  ✓           │    │
│  │ 얼굴 인증 완료           │    │
│  └─────────────────────────┘    │
├─────────────────────────────────┤
│  현재 임무                       │
│  ┌─────────────────────────┐    │
│  │ [대기중] 김철수 고객 배송   │    │
│  │ 서울시 강남구 역삼동 123   │    │
│  │                          │    │
│  │ 메모: 부재시 경비실 맡김   │    │
│  └─────────────────────────┘    │
├─────────────────────────────────┤
│  ┌────────────┐ ┌────────────┐  │
│  │  임무기록   │ │   완료     │  │
│  └────────────┘ └────────────┘  │
└─────────────────────────────────┘
```

#### 임무기록 버튼 동작
1. 버튼 클릭 → 미디어 목록 바텀시트 표시
2. 목록에서 사진/영상 썸네일 클릭 → MediaViewerScreen 팝업
3. 새 기록 추가 버튼 (카메라/갤러리 선택) - 추후 구현
